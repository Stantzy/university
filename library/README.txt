Вам нужно реализовать класс библиотека. 
В библиотеке хранится разная литература: книги и журналы. 
У книг есть автор, заголовок, год выпуска, жанры.
У журналов есть год выпуска, название и тематика.
В библиотеке литература отсортирована по году выпуска.
Вам нужно реализовать методы добавления  литературы в библиотеку, поиск литературы промежутку времени, в который она вышла и метод получения всех книг или журналов из библиотеки.
Напишите юнит тесты для этих методов.
